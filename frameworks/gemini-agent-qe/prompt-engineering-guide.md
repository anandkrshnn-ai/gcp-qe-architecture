# Prompt Engineering for QE Agents (2026 Best Practices)

### Core Principles
- Be specific about role ("You are a senior GCP QE Architect...")
- Force structured output (JSON mode)
- Use low temperature (0.0 - 0.2) for analysis
- Provide clear context limits
- Always ask for confidence score
